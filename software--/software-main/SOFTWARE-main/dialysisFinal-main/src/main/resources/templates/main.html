식단 관리 온라인 쇼핑몰

맞춤형 식단 추천 AI 사용법

1. 식단 목표를 알려주세요.

예: "체중 감량을 원합니다.", "근육량을 늘리고 싶어요.", "당뇨병 관리에 도움을 받고 싶습니다."

2. 목표를 구체적으로 설정할수록 좋습니다. 자세히 설명할수록 더 맞춤화된 식단을 제공받을 수 있습니다.

예: "2개월 안에 5kg을 감량하고 싶습니다.", "3개월 후 대회를 준비 중이라 체지방률을 5% 줄이고 싶어요."
세부적인 요청사항도 추가할 수 있습니다.

3. 특별히 고려해야 할 사항을 알려주시면, 식단에 반영됩니다.

예: "특정 음식에 알레르기가 있어요." (예: 견과류, 유제품 등), "이 식재료를 포함해 주세요." (예: 닭가슴살, 브로콜리 등), "아침 식사는 제외하고 싶어요.", "채식 위주의 식단을 원합니다."
